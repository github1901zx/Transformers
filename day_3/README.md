# День 3: Механизм внимания (Attention) и визуализация

## 1. Что такое Self-Attention?

**Self-Attention (Самовнимание)** — это ключевой механизм трансформеров, который позволяет модели определять, на какие части входной последовательности нужно обратить внимание при обработке конкретного токена.

Для каждого токена вычисляются три вектора:
- **Query (Запрос):** "Что я ищу?"
- **Key (Ключ):** "Что я могу предложить?"
- **Value (Значение):** "Какую информацию я несу?"

Оценка внимания (Attention Score) вычисляется как скалярное произведение Query одного токена и Key всех остальных токенов, после чего применяется Softmax для получения распределения вероятностей.

## 2. Multi-Head Attention

В трансформерах используется **многоголовое внимание**. Это означает, что механизм внимания запускается несколько раз параллельно с разными весами. Каждая "голова" (head) может фокусироваться на разных аспектах: одна на грамматике, другая на семантических связях, третья на знаках препинания.

## 3. Визуализация: Heatmaps

Матрица внимания показывает силу связи между каждой парой токенов в предложении. 
- Ось X: Токены, на которые смотрят (Keys).
- Ось Y: Текущий токен (Query).
- Значения: Веса внимания (от 0 до 1).

Визуализация в виде тепловой карты (heatmap) позволяет наглядно увидеть, как модель "понимает" структуру предложения.
