# День 5: Fine-tuning модели Transformer

Сегодня мы перешли от использования трансформера как экстрактора признаков к полноценному дообучению (Fine-tuning).

## Содержание

1. **Подготовка данных:** Создание PyTorch `Dataset` и `DataLoader` для эффективной подачи данных в модель.
2. **Fine-tuning:** Обучение всех весов модели `DistilBERT` под конкретную задачу классификации.
3. **Оценка и сохранение:** Сохранение обученной модели и её метрик.

## Запуск

```bash
python3 day_5/fine_tuning.py
```

## Основные этапы реализации

### Задача 6: Функции для обучения и оценки
Реализованы функции `train_epoch` (проход по данным с обновлением весов) и `evaluate` (подсчет Accuracy и Macro F1).

### Задача 7: Обучение модели
Модель обучается на 3 эпохи с использованием оптимизатора `AdamW`. Благодаря дообучению весов самого трансформера, модель достигает высокой точности.

### Задача 8: Сохранение модели
Модель и токенизатор сохраняются в папку `fine_tuned_model` с помощью метода `.save_pretrained()`, что позволяет легко загрузить их позже. Метрики записываются в `fine_tuned_results.txt`.
