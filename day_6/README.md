# День 6: Инференс и сравнение моделей

Сегодня мы провели сравнение между базовой моделью (Baseline) и дообученной моделью (Fine-tuned).

## Содержание

1. **Загрузка моделей:** Загрузка `DistilBERT` (fine-tuned) и Логистической регрессии на эмбеддингах (baseline).
2. **Функции предсказания:** Реализация инференса для обоих подходов с обработкой вероятностей.
3. **Сравнение:** Тестирование моделей на новых примерах и анализ их согласованности.
4. **Матрица ошибок:** Визуализация качества классификации на тестовом наборе данных.

## Запуск

```bash
python3 day_6/inference_comparison.py
```

Скрипт загружает ранее сохранённые артефакты из `fine_tuned_model/` и `baseline_model/`. Если чекпоинты отсутствуют, выдаёт ошибку — обучение не запускается.

## Основные результаты

*   **Fine-tuned модель:** Демонстрирует высокую уверенность в предсказаниях (вероятности близки к 0 или 1).
*   **Baseline модель:** Показывает аналогичную точность на данном небольшом наборе данных, используя эмбеддинги `[CLS]` токена.
*   **Сравнение:** Обе модели успешно справились с классификацией тестовых примеров, таких как "This movie was absolutely fantastic!" (Положительный) и "Boring and too long" (Отрицательный).
