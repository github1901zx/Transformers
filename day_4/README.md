# День 4: Baseline без обучения трансформера

Этот день посвящен созданию классификатора текстов на основе предобученных трансформеров без их дообучения (Fine-tuning). Мы используем модель как экстрактор признаков.

## Содержание

1. **Токенизация:** Использование `AutoTokenizer` для подготовки текстов.
2. **Извлечение эмбеддингов:** Получение векторов `[CLS]` токена из последнего скрытого слоя модели.
3. **Классификация:** Обучение логистической регрессии на полученных векторах.

## Запуск

```bash
python3 day_4/baseline_transformer.py
```

## Основные этапы реализации

### Задача 1: Токенизация
Функция `tokenize_texts` преобразует список строк в тензоры PyTorch, добавляя padding и выполняя truncation до указанной длины.

### Задача 2: Извлечение CLS-эмбеддингов
Функция `get_cls_embeddings` прогоняет тексты через модель `distilbert-base-uncased` и извлекает первый вектор из последовательности (`[CLS]`), который обычно используется как агрегированное представление всего предложения.

### Задача 3: Логистическая регрессия
Мы используем `scikit-learn` для разделения данных и обучения классификатора. Результаты (метрика Macro F1 и отчет) сохраняются в файл `day_4/baseline_results.txt`.
